#AI Resume Analyzer
An AI-powered Resume Analyzer built using Python, Streamlit, and PyPDF2
##Features
-Upload PDF Resume 
-Extract Resume Text
-Detect Skills
-Resume Score Calculation 
-Recommended Skills
-Resume Quality Analysis
##Technologies Used
-Python
-Streamlit
-PyPDF2
-Git
-GitHub
##Project Demo 
Upload your resume and get:
-Resume Score
-Skill Detection
-Improvement Suggestions
